# Notes App

A simple and elegant Flutter notes application with local storage using Hive database.

## Features

- **Create Notes** - Add new notes with title and content
- **Edit Notes** - Tap on any note to modify it
- **Delete Notes** - Remove individual notes or bulk delete multiple selected notes
- **Search** - Real-time search through note titles and content
- **Multiple Selection** - Select multiple notes with checkboxes and "select all" functionality
- **Local Storage** - Notes persist between app restarts using Hive database
- **Czech Localization** - Full UI in Czech language
- **Note Counter** - View total number of notes in the app bar
- **Timestamps** - Automatic creation and update timestamps for each note
